SocialSharing Plugin Demo

The SocialSharing plugin allows you to share text, a file, or a URL (or all three) via the native sharing widget of your device.

You can share an image, pdf, or any other file from anywhere on the internet, the local filesystem, or the www folder of your app.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/Windows Phone - App package.
